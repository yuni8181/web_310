HTML 학습서

HTML /CSS /JAVASCRIPT 의 학습내용을 첨부한다.
  • HTML
  • CSS
  • JAVASCRIPT

html이란? Hyper text markup language 로 웹페이지를 읽을수 있게 만들어주는 언어이다.

  • 제목태그와글자태그
  • 텍스트태그
  • div태그/span태그
  • a태그/img태그
  • ol/ul태그
  • table태그
  • 시맨특태그종류

CSS란? Cascading style sheet 의 약자로 HTML 페이지 스타일을 지정해 주는 스타일 시트 이다.

자바스크립트는 HTML 페이지에서 사용자의 반응등을 처리하는 스크립트작성언어이다.
