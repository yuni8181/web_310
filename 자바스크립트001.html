자바스크립트란?

다양한 객체들로 구성되어 있으며 메서드를 통해 제어하는 객체기반언어

객체와 메서드의 의미

스마트폰,전화해라() -> 스마트폰(객체) , 전화해라(메서드) 메서드뒤에 ()괄호는 매개변수

주요단어

  • 변수 - 변화하는값 / 선언문을 통해 제어된다. var,let,const
  • 함수 - 계산식을 만든다. function(){ 계산식 }
  • 조건문 - if문 : 만일 ~~~~ 한다면 ~~~이어라 if(money > 3000){ eatkim() } else { nothing()}
  • 반복문 - 구구단 처럼 일정한 공식을 반복 출력하는것 for문 / while문
