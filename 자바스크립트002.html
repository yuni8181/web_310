연산자란?

변수로 만들어둔 대상이 되는 데이터로 산술연산자,비교연산자,대입연산자,증감연산자 등이 있다.

산술연산자

  • 더하기 +
  • 빼기 -
  • 곱하기 *
  • 나누기 /
  • 나머지 %
예제는 콘솔박스에서 확인가능

비교연산자

  • a는 b보다 크다 a>b
  • a는 b보다 크거나 같다 a>=b
  • a는 b보다 작다 a < b
  • a는 b보다 작거나 같다 a <= b
  • a와 b는 같다 a == b
  • a와 b는 다르다 a != b
예제는 콘솔박스에서 확인가능

대입연산자

  • 더해서대입해라 a+=b -> a=a+b
  • 빼서대입해라 a-=b -> a=a-b
  • 곱해서대입해라 a*=b -> a=a*b
  • 나눠서대입해라 a/=b -> a = a/b
  • 나머지를대입해라 a%=b -> a=a%b
예제는 콘솔박스에서 확인가능

증감연산자

  • 1씩 더하기 ++
  • 1씩 빼기 --
예제는 콘솔박스에서 확인
